# Contributing

Thanks for your interest in HODL your herb!

## Scope

This repository hosts the **artwork** for the HODL your herb / WeedSigner
rolling-paper booklet. Contributions are welcome for:

- Fixes to source files (typos, alignment, color profiles)
- Additional export formats or printer-specific variants
- Translations of on-pack text

## Workflow

1. Fork the repo and create a feature branch.
2. Keep source files in `source/`, exports in `print-ready/`, previews in `preview/`.
3. Open a pull request describing the change and which edition it affects.

## Asset conventions

- **Source:** editable vector formats (SVG preferred; AI/EPS accepted).
- **Print-ready:** PDF with correct trim size, bleed and any foil/spot layers.
- **Preview:** PNG or JPG, RGB, web resolution.
